# Sensoriamento-PR-Gama-2025.2

## What is about?
Image Processing for the detection of agricultural areas and native vegetation

## Repo Structure

```
.
├── LICENSE
├── README.md
└── training_codes
    ├── build/
    ├── CMakeLists.txt
    ├── compile_commands.json -> build/compile_commands.json
    ├── README.md
    ├── scripts/
    └── src/
```

### Files and Dirs

This is how files and directories are organize.

- `build/`: directory for building process by cmake;
- `CMakeLists.txt`: configuration file for cmake;
- `compile_commands.json`: json file for LSP;
- `scirpts/`: scirpts file to help with cmake;
- `src/`: main files
